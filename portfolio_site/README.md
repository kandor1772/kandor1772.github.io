# Portfolio Site

Files:

- `index.html`
- `styles.css`
- `config.js`
- `.nojekyll`
- `GITHUB_PAGES.md`

Fastest edit path:

1. Open `config.js`
2. Replace `href: "#"` placeholders with your real links
3. Change titles and body copy
4. Open `index.html` in a browser

Best use:

- publish directly on GitHub Pages
- or embed/link it from your Google Site
- or upload the whole folder to any static host later
